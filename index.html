Surrounded Regions 解题报告

发表于 2017-06-01 | 分类于 interview

[Problem]

Given a 2D board containing ‘X’ and ‘O’, capture all regions surrounded by ‘X’.
A region is captured by flipping all ‘O’’s into ‘X’’s in that surrounded region.

Analysis

问题 理解与解答
概括 找出了边界相关以外的区域并填充.
思路 遍历边界,通过递归的方式找到边界相关区域并标记,整体遍历填充其他区域并将标记取消.
提示 开始时只想到遍历边界并将判断方法提出来,后来发现这个方法就是递归判断.
复杂度 O(n*m): O(1) – 直接修改目标数组:

/** java
Surrounded Regions

Given a 2D board containing ‘X’ and ‘O’, capture all regions surrounded by ‘X’.

A region is captured by flipping all ‘O’’s into ‘X’’s in that surrounded region.

分析:
一句话概括这个题要你干嘛:
找出了边界相关以外的区域并填充.
思路:
遍历边界,通过递归的方式找到边界相关区域并标记,整体遍历填充其他区域并将标记取消.
有哪些条件提示我想到了这个题的解法:
开始时只想到遍历边界并将判断方法提出来,后来发现这个方法就是递归判断.
时间复杂度: O(n*m):，
空间复杂度: O(1) – 直接修改目标数组:

public class Solution {
/*

 * @param board a 2D board containing 'X' and 'O'
 * @return void
 */
int m;
int n;
int[] dx = {1,-1,0,0};
int[] dy = {0,0,1,-1};
public void surroundedRegions(char[][] board) {
    if(board == null || board.length <= 2 || board[0].length <= 2){
        return;
    }

    m = board.length;
    n = board[0].length;
    for(int i = 0; i < m; i++){
        change(board,0,i);
        change(board,m-1,i);
        change(board,i,0);
        change(board,i,n-1);
    }
     for(int i = 0; i < m; i++){
          for(int j = 0; j < n; j++){
              if(board[i][j] == 'O'){
                  board[i][j] = 'X';
              }else if(board[i][j] == 'Y'){
                  board[i][j] = 'O';
              }
          }
     }
}
void change(char[][] board,int x,int y){
    if(x < 0 || x > m-1 || y < 0 || y > n-1){
        return;
    }
    if(board[x][y] == 'O'){
        board[x][y] ='Y';
        for(int i = 0; i < 4; i++){
            change(board,x+dx[i],y+dy[i]);
        }
    }
}

}

Reverse Linked List II 解题报告

发表于 2017-05-05 | 分类于 interview

Problem

Reverse a linked list from position m to n.

Example:
Given 1->2->3->4->5->NULL, m = 2 and n = 4, return 1->4->3->2->5->NULL.

Analysis

问题 理解与解答
概括 反转指定位置之间的节点
思路 1.逐个反转; 2.选取指定区域,先整体反转再拼接
提示 1.反转linkedList; 2.复杂度要求in place, one time.
复杂度 O(n), in place

类似题目 : Reverse Linked List;

总结:

  • 开始判断 head == null || head.next ==null
  • dummyNode
  • 逐个反转?
  • m, n的值是从1开始的.(而不是从0开始)
  • 当反转一个点后,因头节点相对位置变化,需要用pre重新赋值, 否则只能在前两个点间打转, 直到内存溢出.
