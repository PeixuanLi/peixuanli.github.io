Angular组件间通讯

输入属性 (父组件对象输出至子组件)

子组件中 @Input() 注释的对象不会影响父组件的值

  • 子组件中:

    • 通过ts文件中的 @Input() 使之变为可被输入的对象

  • 父组件中:

    • html的子组件标签中的通过[属性绑定]将父组件中的变量绑定至子组件

输出属性 (子组件对象输出至父组件)

  • 子组件中:

    • 声明为 @Output 的 EventEmitter 对象, 类型是带输出的对象类, 和待输出对象的名称或别名

    • html的子组件标签中写对象名称和处理方法

    • 在ts文件中写处理方法

[Problem]

Given the root and two nodes in a Binary Tree. Find the lowest common ancestor(LCA) of the two nodes.
The lowest common ancestor is the node with largest depth which is the ancestor of both nodes.
Notice : Assume two nodes are exist in tree.
Example: For the following binary tree:
4
/ \
3 7
/ \
5 6
LCA(3, 5) = 4
LCA(5, 6) = 7
LCA(6, 7) = 7

Analysis

问题 理解与解答
概括 求最近祖先
思路 divide & conquer
提示 要把返回值与其本身的意义分开.
复杂度 O(n)

类似题目 :

总结:

  • 要分析出方法中返回值怎么表示你要的结果, 是否包含所有的情况
  • 返回值可以是: null, A, B, any child node, current node
  • 对应的情况是:
  • null, AB均不在其子树中
  • A, 遇到A无论是否遇到B
  • B, 遇到B无论是否遇到A
  • any child node, 有子树已经包含A与B
  • current node, 左右分别是A 和 B

Surrounded Regions 解题报告

发表于 2017-06-01 | 分类于 interview

[Problem]

Given a 2D board containing ‘X’ and ‘O’, capture all regions surrounded by ‘X’.
A region is captured by flipping all ‘O’’s into ‘X’’s in that surrounded region.

Analysis

问题 理解与解答
概括 找出了边界相关以外的区域并填充.
思路 遍历边界,通过递归的方式找到边界相关区域并标记,整体遍历填充其他区域并将标记取消.
提示 开始时只想到遍历边界并将判断方法提出来,后来发现这个方法就是递归判断.
复杂度 O(n*m): O(1) – 直接修改目标数组:

类似题目 :
Number of Islands, Number of Islands II

总结:

/** java
Surrounded Regions

Given a 2D board containing ‘X’ and ‘O’, capture all regions surrounded by ‘X’.

A region is captured by flipping all ‘O’’s into ‘X’’s in that surrounded region.

分析:
一句话概括这个题要你干嘛:
找出了边界相关以外的区域并填充.
思路:
遍历边界,通过递归的方式找到边界相关区域并标记,整体遍历填充其他区域并将标记取消.
有哪些条件提示我想到了这个题的解法:
开始时只想到遍历边界并将判断方法提出来,后来发现这个方法就是递归判断.
时间复杂度: O(n*m):，
空间复杂度: O(1) – 直接修改目标数组:
有哪些我做过的题目:
Number of Islands, Number of Islands II

/
public class Solution {
/*

 * @param board a 2D board containing 'X' and 'O'
 * @return void
 */
int m;
int n;
int[] dx = {1,-1,0,0};
int[] dy = {0,0,1,-1};
public void surroundedRegions(char[][] board) {
    if(board == null || board.length <= 2 || board[0].length <= 2){
        return;
    }

    m = board.length;
    n = board[0].length;
    for(int i = 0; i < m; i++){
        change(board,0,i);
        change(board,m-1,i);
        change(board,i,0);
        change(board,i,n-1);
    }
     for(int i = 0; i < m; i++){
          for(int j = 0; j < n; j++){
              if(board[i][j] == 'O'){
                  board[i][j] = 'X';
              }else if(board[i][j] == 'Y'){
                  board[i][j] = 'O';
              }
          }
     }
}
void change(char[][] board,int x,int y){
    if(x < 0 || x > m-1 || y < 0 || y > n-1){
        return;
    }
    if(board[x][y] == 'O'){
        board[x][y] ='Y';
        for(int i = 0; i < 4; i++){
            change(board,x+dx[i],y+dy[i]);
        }
    }
}

}
