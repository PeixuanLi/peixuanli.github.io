Lowest Common Ancestor解题报告

发表于 2017-06-07 | 分类于 interview

[Problem]

Given the root and two nodes in a Binary Tree. Find the lowest common ancestor(LCA) of the two nodes.
The lowest common ancestor is the node with largest depth which is the ancestor of both nodes.
Notice : Assume two nodes are exist in tree.
Example: For the following binary tree:
4
/ \
3 7
/ \
5 6
LCA(3, 5) = 4
LCA(5, 6) = 7
LCA(6, 7) = 7

Analysis

问题 理解与解答
概括 求最近祖先
思路 divide & conquer
提示 要把返回值与其本身的意义分开.
复杂度 O(n)

总结:

  • 要分析出方法中返回值怎么表示你要的结果, 是否包含所有的情况
  • 返回值可以是: null, A, B, any child node, current node
  • 对应的情况是:
  • null, AB均不在其子树中
  • A, 遇到A无论是否遇到B
  • B, 遇到B无论是否遇到A
  • any child node, 有子树已经包含A与B
  • current node, 左右分别是A 和 B
