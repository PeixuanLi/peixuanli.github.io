排序要点:

名称 大体顺序 复杂度 额外空间 稳定性
Quick Sort 先整体后局部有序 分O(n),治O(1), 共O(nlgn) 不需要 不稳定
Merge Sort 先局部后整体有序 分O(1), 治O(n),共O(nlgn) 需要 稳定

注:

  1. Quick Sort 最坏情况:N^2; 最好情况:nlgn; 平均复杂度O(nlgn)
  2. Quick Sort优于Merge Sort, 不需要额外空间.
  3. Merge Sort 不会影响相等情况的顺序.
